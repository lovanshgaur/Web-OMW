# web-omw  
> Generate ready-to-use web project templates in seconds.

## 🚀 What is this?

`web-omw` is a lightweight CLI tool that helps you start building websites instantly — no setup, no frameworks, no unnecessary complexity.

Run one command, pick a template, and get a clean project structure with HTML, CSS, and JavaScript — ready to go.

---

## ✨ Features

- ⚡ Create a new project instantly  
- 🎯 Choose from multiple ready-to-use templates  
- 🪶 Lightweight — uses external assets (no heavy files)  
- 🧠 No frameworks — full control over your code  
- 🛠 Perfect for prototypes, landing pages, and small projects  

---

## 📦 Usage

No installation needed:

```bash
npx web-omw
```

Follow the prompts:

1. Enter your project name  
2. Select a template  
3. Project is generated instantly  

Then:

```bash
cd your-project-name
```

Open `index.html` in your browser and start building.
